Complete Store Management App is a full-stack web application designed to manage retail store operations digitally.
The system allows store owners to manage products, track inventory, handle sales transactions, and perform CRUD operations through a user-friendly interface connected to a Java-based backend.

The project demonstrates understanding of multi-tier architecture, RESTful communication, and frontend-backend integration.
